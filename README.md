# Jinmori
Jinmori is a portable package manager for Standard ML, focusing on MLton support (for now).

## Dependencies
**None**, if you are running a UNIX-based system like Linux or MacOS.
Jinmori only uses POSIX shell constructs, so not even `bash` or `zsh` is required.

Running the Jinmori test suite requires [try](https://github.com/binpash/try).

## Setup
Jinmori aims for a seamless setup.
Simply download the script and run the setup command like below.
```sh
wget https://raw.githubusercontent.com/Forthoney/jinmori/refs/heads/main/jinmori
chmod +x jinmori # allow jinmori to be run on your computer
./jinmori setup
# Restart the shell session (close your terminal and reopen it)

jinmori # This should now output a help page
```

## Deletion
Jinmori should only every touch two locations on the file system:
It will append to the existing shell profile script, whose path is shell dependent,
and create a new directory for Jinmori, which is `~/.jinmori/` by default.

To delete Jinmori,
remove the Jinmori autogenerated path modification code from the shell profile script
and recursively delete the Jinmori home directory (`rm -rf $HOME/.jinmori`)
